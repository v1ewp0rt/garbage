Este código fue escrito manualmente por el autor y se encarga de las siguientes operaciones:
  - Carga equidistante de contraseñas de la colección
  - Cálculo de probabilidades secuenciales
  - Generación de archivo TXT de contraseñas seleccionadas
  - Generación de archivo DAT con matriz para el Mapa de calor
  - Recolección de características para cada contraseña cargada
  - Cálculo de Información propia condicional promedio
  - Generación de archivos DAT para Diagramas de barras
  - Generación de archivos DAT para Histogramas, Box Plot y Dispersogramas
  - Generación de archivos CSV con tablas para el análisis descriptivo
  - Generación de archivo CSV de correlaciones para las tablas de regresión
  - Generación de dataset CSV completo
  - Demostración interactiva de la IPCP para contraseñas ingresadas manualmente
